# Activity Module

This document outlines the Activity Timeline & Event Center.

## 1. Unified Event Center

- Maps and aggregates recent job transition states from `/dashboard/activity` and `/dashboard/timeline`.
- Renders an audit log tracking state paths chronologically.

---

## 2. Event Filters

- Supports filtering events by categories (Job submitted, queued, claimed, running, completed, failed, retry pending).
- Supports keyword searches on event payloads.
