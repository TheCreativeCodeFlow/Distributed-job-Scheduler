# DLQ Module

This document outlines the Dead Letter Queue (DLQ) Management module.

## 1. Dead Letter Entries Telemetry

- Lists quarantined entry IDs, original job reference links, targeted queues, and exact failure reason messages.
- Filters entries by queue name, ACTIVE/REPLAYED status, or search keywords.

---

## 2. Details Inspection

- Displays exit status codes, stack traces, parameters metadata, and original enqueued payloads.
- Offers manual operator control action panels.
