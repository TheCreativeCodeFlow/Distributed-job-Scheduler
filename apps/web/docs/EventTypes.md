# Event Types

This document maps event identifiers to icons and descriptions.

- **job.submitted**: FileCode icon. Indicates a job has been created.
- **job.queued**: Database icon. Indicates queue enqueue.
- **job.running**: Play icon. Active execution.
- **job.completed**: ShieldCheck icon. Execution success.
- **job.failed**: AlertTriangle icon. Execution exception.
- **job.retry_pending**: GitCommit icon. Waiting for backoff.
- **job.retry_exhausted**: Skull icon. Quarantined.
