# Job Management Module

This document outlines the Job Management module interface.

## 1. Job Telemetry Directory

- Displays active scheduler jobs including: Job ID, Queue, Project, Organization, Status, Priority, Attempts, and Duration metrics.
- Local parameters filter search text inputs against targets and dropdown state mappings.

---

## 2. Job Details view

- Highlights: complete job metadata parameters, read-only code JSON editors, worker references, attempts executions, and state-aware operations buttons.
- Features copy triggers to quickly capture full system IDs.
