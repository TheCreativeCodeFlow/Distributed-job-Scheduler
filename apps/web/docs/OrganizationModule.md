# Organization Management Module

This document outlines the organization administration modules details.

## 1. Directory List

- **Organizations Index**: Displays active/suspended tenant structures, member totals, and project configurations inside a sortable `DataTable`.

---

## 2. Details Dashboard

- Renders organization-specific telemetry data.
- Projects table maps linked project entities.
- Chronological logs logs activity records.
- Action menus route directly to settings panel keys.
