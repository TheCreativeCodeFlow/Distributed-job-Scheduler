# Operations Overview Dashboard

This document details the layout structure of the home Operations Overview page.

## 1. Top Section

- **Welcome Banner**: Displays the active organization context selection, current project, and quick refresh triggers.
- **KPI Metrics Grid**: Houses 8 statistics cards mapping count totals.

---

## 2. Main Visual layout

- **Left Column**: System health status indicators and Live activity scroll feeds.
- **Right Column**: Quick actions triggers and job throughput donut distributions.
- **Auxiliary Bottom Row**: Radial donut charts breaking down Queue Activity status levels and Worker states.
