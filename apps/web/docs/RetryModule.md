# Retry Module

This document outlines the Retry Management dashboard.

## 1. Retry Telemetry Console

- Lists overall statistics parameters: success rates, total attempts counters, and exhausted totals.
- Filters by active queue target, retry running status, or search query criteria.

---

## 2. Details config parameters

- Displays backoff calculation strategy (EXPONENTIAL, FIXED, LINEAR).
- Displays active remaining countdown delays before next automatic execution.
- Lists execution history failures.
