# Scheduled Jobs Module

This document outlines the Scheduled Jobs module dashboard interface.

## 1. Scheduled Telemetry Directory

- Displays active scheduler scheduled jobs including: Scheduled ID, Promoted Job ID, Queue, status, and remaining countdown times.
- Integrates local selectors matching project and queue filters.

---

## 2. Details view

- Displays payload and metadata JSON objects.
- Displays cron expressions and execution time parameters.
