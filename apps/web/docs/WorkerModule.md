# Worker Management Module

This document outlines the Worker Management module operations dashboard.

## 1. Directory Telemetry Console

- Lists hostname, status state indicators, capacity slots meters, heartbeat logs, and version attributes.
- Filters by active queue target, running status, or custom metadata tag criteria.

---

## 2. Capacity Bars & Active Claims

- Displays concurrency slot usage graphs detailing allocations.
- Displays claims lists showing lease intervals.
